EmecoはCocoaAPIとVimのキーバインドで操作できるプレビュー機能付きエディターです。

HTMLとCSSのコードは書き換えた時点で即時反映されます。

JavaScriptのコードも実行できます。

HTMLとCSSはEmmetに対応しています。

代表的なキーバインド

    <C-t>: エディタ切り替え
    <C-r>: JavaScript実行
    <C-s>: 保存
    <C-y>: 補完
    <Cmd-k>: HTMLタグを消す
    Tab: Emmet展開（HTML・CSS）
